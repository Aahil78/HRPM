# HRPM - A High Speed PID Based Line Follower
A very fast Line Following Robot with a cusom IR Array and control PCB, uses PID Based algorithim.

## Features:
  - High Speed for fast track completion
  - Custom 20 Sensor IR Array Module for precision
  - Integrated MCU, Motor Driver, and LiPo System (Charger Seperate) For good control and OTA PID Tuning -- COMING SOON
  - Custom 3D Printed Chassis for modularity. -- COMING SOON
  - More to come soon!

## Why I made this:

I made this because I go to comps often and many times I lose, I've noticed that many bots aren't fast so I decided; what if I make a high speed LFR, that'll help me win comps and get some prizes and network around with people.
